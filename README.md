# Digital-Hand-Sanitizer-Dispenser-using-a-servo-motor-and-an-ultrasonic-sensor
Automatic hand sanitizer dispenser using Arduino UNO, HC-SR04 ultrasonic sensor, and a servo motor. When a hand is detected within 20cm, the servo rotates 90° to dispense sanitizer and resets automatically. Built and simulated on Tinkercad. No contact needed — fully touchless and automatic.
